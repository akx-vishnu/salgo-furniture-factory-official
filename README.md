# Salgo Furniture Factory

A premium, dark-themed **Quiet Luxury** furniture factory website. Designed for a high-end, sophisticated user experience using modern web technologies.

![Salgo Furniture Design](https://via.placeholder.com/1200x600.png?text=Salgo+Furniture+Factory+Preview)

## 🛠 Tech Stack

- **Framework:** [React 19](https://react.dev/)
- **Build Tool:** [Vite 8](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation:** [Framer Motion v12](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Smooth Scroll:** [Lenis](https://github.com/darkroomengineering/lenis)
- **Language:** TypeScript

## ✨ Key Features

- **Quiet Luxury Aesthetic:** Dark-themed, high-contrast design with a premium feel.
- **Glassmorphism Nav:** Sophisticated navigation with blur effects.
- **Micro-animations:** Smooth reveals and hover states using Framer Motion.
- **Performance First:** Optimized with Vite and React 19 concurrent features.
- **Custom Design System:** Tailored color palette (`cream`, `charcoal`, `gold`, `wood`).

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/akx-vishnu/salgo-furniture-factory.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗 Project Structure

- `src/components/`: Reusable UI components.
- `src/App.jsx`: Main application orchestrator.
- `src/index.css`: Global styles and Tailwind v4 theme configuration.
- `public/`: Static assets.

## 📜 Scripts

- `npm run dev`: Start development server.
- `npm run build`: Build for production (includes typechecking).
- `npm run preview`: Locally preview production build.
